* 插入连接
```
[链接文字](链接网址 "标题")
比如
[Home](index.md "返回 Home")
```
>[Home](index.md "返回 Home")

* 插入图片
```markdown
![blockchain](img/i.jpeg "logo")
```

* 插入代码

```python
import sys
def do_f():
    sys.exit()
```
